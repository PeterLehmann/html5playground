This page summarizes the skills measured for the 70-480 exam.

A Implement and Manipulate Document Structures and Objects (24%)
  1. Create the document structure.

    This objective may include but is not limited to: structure the UI by using semantic markup, including for search engines and screen readers (Section, Article, Nav, Header, Footer, and Aside); create a layout container in HTML

    Sample page(TJK)
  2. Write code that interacts with UI controls. This objective may include but is not limited to: programmatically add and modify HTML elements; implement media controls; implement HTML5 canvas and SVG graphics Canvas sample (TJK)
  3. Apply styling to HTML elements programmatically. This objective may include but is not limited to: change the location of an element; apply a transform; show and hide elements
  4. Implement HTML5 APIs. This objective may include but is not limited to: implement storage APIs, AppCache API, and Geolocation API det er nok nyt for en del af os. jeg har arbejdet lidt med storage API'et, men ellers ikke de andre
  5. Establish the scope of objects and variables. This objective may include but is not limited to: define the lifetime of variables; keep objects out of the global namespace; use the “this” keyword to reference an object that fired an event; scope variables locally and globally det er ren javascript. vi skal bare have styr på de mest almindelige patterns i javascript og der hvor den er mærkelig, især scope og this
  6. Create and implement objects and methods. This objective may include but is not limited to: implement native objects; create custom objects and custom properties for native objects using prototypes and functions; inherit from an object; implement native methods and create custom methods det må være javascript? jeg går ud fra at "native objects" bare betyder at man laver sine egne javascript-objekter. Jeg ved ikke hvad der er forskellen på native methods og custom methods?
B Implement Program Flow (25%)
  1. Implement program flow. This objective may include but is not limited to: iterate across collections and array items; manage program decisions by using switch statements, if/then, and operators; evaluate expressions det er vist lige ud af landevejen. bare de almindelige kontrolstrukturere i Javascript
  2. Raise and handle an event. This objective may include but is not limited to: handle common events exposed by DOM (OnBlur, OnFocus, OnClick); declare and handle bubbled events; handle an event by using an anonymous function
  3. Implement exception handling. This objective may include but is not limited to: set and respond to error codes; throw an exception; request for null checks; implement try-catch-finally blocks det burde også være lige til at gå til
  4. Implement a callback. This objective may include but is not limited to: receive messages from the HTML5 WebSocket API; use jQuery to make an AJAX call; wire up an event; implement a callback by using anonymous functions; handle the “this” pointer og man skal kunne det udenad :-(
  5. Create a web worker process.

    This objective may include but is not limited to: start and stop a web worker; pass data to a web worker; configure timeouts and intervals on the web worker; register an event listener for the web worker; limitations of a web worker

    Example code with a webworker (TJK)
C Access and Secure Data (26%)
  1. Validate user input by using HTML5 elements. This objective may include but is not limited to: choose the appropriate controls based on requirements; implement HTML input types and content attributes (for example, required) to collect user input det er vist mest de nye ting i forms, f.eks. type="email" og den slags
  2. Validate user input by using JavaScript. This objective may include but is not limited to: evaluate a regular expression to validate the input format; validate that you are getting the right kind of data type by using built-in functions; prevent code injection regexp ...
  3. Consume data. This objective may include but is not limited to: consume JSON and XML data; retrieve data by using web services; load data or get data from other sources by using XMLHTTPRequest og man skal kunne huske det ...
  4. Serialize, deserialize, and transmit data. This objective may include but is not limited to: binary data; text data (JSON, XML); implement the jQuery serialize method; Form.Submit; parse data; send data by using XMLHTTPRequest; sanitize input by using URI/form encoding
D Use CSS3 in Applications (25%)
  1. Style HTML text properties. This objective may include but is not limited to: apply styles to text appearance (color, bold, italics); apply styles to text font (WOFF and @font-face, size); apply styles to text alignment, spacing, and indentation; apply styles to text hyphenation; apply styles for a text drop shadow det burde være lige ud af landevejen. Men vi skal have styr på de nye ting, som font-face dropshadows
  2. Style HTML box properties. This objective may include but is not limited to: apply styles to alter appearance attributes (size, border and rounding border corners, outline, padding, margin); apply styles to alter graphic effects (transparency, opacity, background image, gradients, shadow, clipping); apply styles to establish and change an element’s position (static, relative, absolute, fixed) det burde også være lige til at gå til
  3. Create a flexible content layout. This objective may include but is not limited to: implement a layout using a flexible box model; implement a layout using multi-column; implement a layout using position floating and exclusions; implement a layout using grid alignment; implement a layout using regions, grouping, and nesting der er åbenbart nogle helt nye flexboxe og multiple columns i CSS3 som kan bruges til at layoute elementer. Det har jeg slet ikke arbejdet med
  4. Create an animated and adaptive UI. This objective may include but is not limited to: animate objects by applying CSS transitions; apply 3-D and 2-D transformations; adjust UI based on media queries (device adaptations for output formats, displays, and representations); hide or disable controls animationer
  5. Find elements by using CSS selectors and jQuery. This objective may include but is not limited to: choose the correct selector to reference an element; define element, style, and attribute selectors; find elements by using pseudo-elements and pseudo-classes (for example, :before, :first-line, :first-letter, :target, :lang, :checked, :first-child) Simpelt - men det er bare med at huske det ...
  6. Structure a CSS file by using CSS selectors. This objective may include but is not limited to: reference elements correctly; implement inheritance; override inheritance by using !important; style an element based on pseudo-elements and pseudo-classes (for example, :before, :first-line, :first-letter, :target, :lang, :checked, :first-child)
